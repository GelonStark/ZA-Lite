**3.软件的功能说明**

> <font style="color:#8A8F8D;">经过上面的设置后，ZA已经可以开始使用</font>
>

**3.1 选择任务**

选择本次循环执行的任务，开始后会循环直到体力耗尽，在HIA的任务单次按照最大值（100体力）计算，若选择"无"则跳过战斗任务

![](https://cdn.nlark.com/yuque/0/2025/png/55034587/1743261989571-2ddf8624-1ec1-416a-8cc6-c6e822336b76.png)

**3.2 日常活动**

勾选即开启，其中**领奖励**包含：每日 每周 月卡 版本签到以及邮件奖励

![](https://cdn.nlark.com/yuque/0/2025/png/55034587/1743262736749-d42d9c15-3d5e-4199-9cfe-a6c44fc26696.png)

**3.3 运行速度**

包含 默认/快/慢 三挡，正常使用默认即可，低配电脑任务失败尝试慢档，快档有卡住的可能性不推荐使用

![](https://cdn.nlark.com/yuque/0/2025/png/55034587/1743262441365-67102e9b-f6db-4e08-b4ef-21c552fdc69c.png)

**3.4 任务完成后的操作**

任务结束后 无操作/退出游戏/关机，配合windows任务计划实现全自动无人操作

![](https://cdn.nlark.com/yuque/0/2025/png/55034587/1743262562968-b23ba12f-8446-4072-a9e6-d3c6ec6af699.png)

**3.5 挑战等级**

选择是否使用副本可用的最大等级，注意这里的最大指的是游戏副本目前的最大等级，不是指你的账号能打的最大等级，如果还有未解锁的难度不要开启这项功能，默认则跳过（沿用上次的选择）

![](https://cdn.nlark.com/yuque/0/2025/png/55034587/1743529745690-5b043795-e14a-47d0-8efd-dc8cd4ba6ac6.png)

选择要使用的队伍，所有队伍名称应是默认状态不要修改，默认则跳过，编队建议选择无敌帧多的大C，

不要用在前台完全无用的机制复杂角色

**4.你可能需要知道的**

**4.1 强制停止**

在任务程序执行期间（特别是战斗时），如果要介入操作或回到桌面，先按Alt停止程序，避免模拟操作自动乱按

**4.2 重命名**

游戏内一切可以重命名的东西都不要改名，包括编队名、副本名

**4.3 创建桌面快捷方式**

右键点击ZA Launcher，发送到 / 桌面快捷方式，不要直接拖动文件夹内的文件

**4.4 常见故障处理**

程序启动后进入了游戏，但没反应 >>> 检查管理员权限，

程序能自动操作，但到某一步卡住或无限重试 >>> 检查运行日志(logs)，电脑加载速度慢的使用慢档再次尝试，

依然无法解决的，可能是游戏界面发生变动，截图完整错误界面，或复制对应的运行日志反馈到b站
