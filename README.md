1.软件下载和配置
下载然后进行一些系统设置
1.1 ZA如何下载 ？
网盘链接：
下载压缩包后拖出整个文件夹，运行Updater会自动下载部署最新版本，后续的版本更新也是通过此程序获取
1.2 杀毒软件误报
在下载解压时系统自带的Windows Defender有可能会出现这样的安全弹窗，或者视为威胁直接隔离程序的文件
建议在杀软里给ZA所处的文件夹设置排除项/白名单，或者直接关闭杀软，否则可能导致文件残缺无法正常运行
1.2 管理员权限
（已不再需要手动设置）
程序模拟操作需要管理员权限，Launcher和Main这两个文件都需要权限，
启动软件前，先（右键 - 属性 - 兼容性 - 勾选以管理员身份运行）设置权限
注意：如果移动了文件夹路径变更后管理员权限会失效，需要重新勾选！
1.3 指定游戏路径
打开Launcher并从左下角进入设置页面，
在你安装游戏的根目录（右键游戏图标属性 - 打开文件所在的位置），找到并选择ZenlessZoneZero.exe
2.需要确认的游戏内设置
现在打开游戏设置页面
2.1 分辨率设置
游戏分辨率需要设置为 全屏1920 x 1080 比例16:9 关闭HDR/滤镜
如果屏幕比例不支持1080P，需使用ZA Launcher启动游戏 
2.2 字体设置
必须设置为 全局细体
3.软件的功能说明
经过上面的设置后，ZA已经可以开始使用
3.1 选择任务
选择本次循环执行的任务，开始后会循环直到体力耗尽，在HIA的任务单次按照最大值（100体力）计算，若选择"无"则跳过战斗任务
3.2 日常活动
勾选即开启，其中领奖励包含：每日 每周 月卡 版本签到以及邮件奖励
3.3 运行速度
包含 默认/快/慢 三挡，正常使用默认即可，低配电脑任务失败尝试慢档，快档有卡住的可能性不推荐使用
3.4 任务完成后的操作
任务结束后 无操作/退出游戏/关机，配合windows任务计划实现全自动无人操作
3.5 挑战等级
选择是否使用副本可用的最大等级，注意这里的最大指的是游戏副本目前的最大等级，不是指你的账号能打的最大等级，如果还有未解锁的难度不要开启这项功能，默认则跳过（沿用上次的选择）
3.6 预备编队
选择要使用的队伍，所有队伍名称应是默认状态不要修改，默认则跳过，编队建议选择无敌帧多的大C，
不要用在前台完全无用的机制复杂角色
3.7 开始任务
点击开始会根据你之前所配置的内容开始执行任务，你也可以通过Main文件直接启动程序
3.8 任务计划程序
在使用windows任务计划程序实现自动化时，选择Main文件作为被启动的文件，
在起始于输入ZA-Lite文件夹路径（示例蓝色部分，不包含""符号）
4.你可能需要知道的
4.1 强制停止
在任务程序执行期间（特别是战斗时），如果要介入操作或回到桌面，先按Alt停止程序，避免模拟操作自动乱按
4.2 重命名
游戏内一切可以重命名的东西都不要改名，包括编队名、副本名
4.3 创建桌面快捷方式
右键点击ZA Launcher，发送到 / 桌面快捷方式，不要直接拖动文件夹内的文件
4.4 常见故障处理
程序启动后进入了游戏，但没反应 >>> 检查管理员权限，
程序能自动操作，但到某一步卡住或无限重试 >>> 检查运行日志(logs)，电脑加载速度慢的使用慢档再次尝试，
依然无法解决的，可能是游戏界面发生变动，截图完整错误界面，或复制对应的运行日志反馈到b站
