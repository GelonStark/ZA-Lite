## 主要功能

- **日常任务 ：** 喝咖啡、自动战斗清体力、刮刮卡
- **周常任务 ：** 零号空洞（战线肃清、特遣调查）
- **领取奖励 ：** 日周常活跃度、大小月卡、邮件、版本签到活动
- 使用任务计划程序添加 ZA-Lite 可执行程序实现**定期自动**开始任务
- 任务**结束后**自动 退出游戏/关闭电脑

## 注意事项
- 软件仅支持 Windows 10/11 系统
- 必须在 **1920 x 1080** 的游戏分辨率下运行，并且游戏内字体设置为**全局细体** 
- 点击**开始**按钮后，程序会在前台进行操作，如果要中途退出先按下**Alt键**终止程序
