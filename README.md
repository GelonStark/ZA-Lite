# 主要功能

- **日常任务 ：** 喝咖啡、自动清体力、领取奖励、刮刮卡
- **周常任务 ：** 零号空洞（战线肃清、特遣调查）
- 使用任务计划程序添加 ZA-Lite 程序实现定期自动执行
- 任务**结束后**自动 退出游戏/关闭电脑
